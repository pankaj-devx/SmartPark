# SmartPark

SmartPark is a smart parking slot rental and availability platform for discovering, listing, reserving, and managing parking spaces.

This project is being built incrementally in engineering phases. Phase 1 defines the product and technical architecture. Phase 2 will scaffold the MERN repository.

## Current Phase

Phase 1: Project architecture and planning

## Planned Stack

- React + Tailwind CSS
- Node.js + Express
- MongoDB + Mongoose
- JWT authentication
- Leaflet first, with optional Google Maps upgrade
- Vercel frontend deployment
- Render backend deployment
